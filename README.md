# HR Analytics Dashboard – Power BI

This project was developed as part of the 4 Week **Microsoft Elevate Program** in collaboration with **AICTE**.

## Project Overview

The dashboard analyzes HR data to provide insights into employee distribution, salary trends, and workforce demographics.

## Tools Used

* Microsoft Power BI
* Data Visualization
* HR Dataset

## Key Features

* Total Employees KPI
* Average Salary KPI
* Department-wise employee distribution
* Age group analysis
* Country-wise employee distribution
* Hiring trend analysis
